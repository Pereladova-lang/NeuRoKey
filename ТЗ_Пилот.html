🧠 NeuRoKey Пилот v1

Техническое задание — пилотная веб-версия для детей 11–14 лет

📅 15 июля 2026 ✅ Утверждено 🌿 feature/pilot-v1

1. Цель и объём

Веб-приложение-тренажёр для проверки главной гипотезы: интересны ли трёхслойные нейрокогнитивные упражнения детям 11–14 лет и видят ли родители ценность в отчётах.

✓ Детский интерфейс ✓ Родительский портал (1 страница) ✓ Подписка ЮKassa + триал 3 дня

✗ Блокировка контента ✗ ИИ-анализ ✗ Чат-бот (вместо него маскот) ✗ Мобильные приложения

2. Упражнения (3 типа, трёхслойные)

ТипАкадемический слойКогнитивный слойМеханика
🕵️ Комикс-детективСмысловое чтениеЛогика, прогнозированиеКомикс 4–6 панелей, часть реплик скрыта — выбрать верную из 3
📊 Аналитик данныхМатематика, статистикаКритическое мышлениеГрафик + вопросы «что можно / нельзя заключить»
🤖 Робо-логикаОсновы алгоритмовПланирование, контроль импульсовКоманды для робота через поле с ловушками; энергия ограничена
  • Контент: рукописные JSON-пулы — 10 вариантов × 3 уровня × 3 типа = 90 записей
  • Сессия = 3 упражнения (~2 мин каждое), итого 5–7 минут — жёсткий потолок
  • Ошибка ребёнка: подсказка + повтор. Никаких красных крестов и штрафов
  • Никаких лидербордов и сравнений с другими детьми

3. Адаптивность (rule-based, без ИИ)

  • Каждый результат пишет очки в 4 шкалы: внимание, память, логика, контроль
  • Ротация: тип вчерашнего первого упражнения не ставится первым сегодня
  • Сложность: точность >80% три раза подряд → уровень +1; <50% → уровень −1
  • Алгоритм прозрачен и объясним родителю (триггер доверия)

4. Мотивация

  • 🔥 Стрик (дни подряд с сессией), бейджи за серии
  • Уровни маскота: «Новичок» → «Мастер фокуса»
  • Маскот со статичными репликами (без LLM): приветствие, похвала, мягкая реакция
  • Позитивное подкрепление, ноль наказаний

5. Подписка (ЮKassa)

ПараметрЗначение
Триал3 дня полного доступа без карты (конфиг в БД, для A/B 1 vs 3 дня)
Тариф299 ₽/месяц, рекуррентный (автосписание)
Free tier после триала1 упражнение в день
Статусыtrial → active → past_due (3 дня грейс) → expired / canceled
ОтменаВ один клик в родительском портале
Webhookspayment.succeeded / payment.canceled + fallback-сверка при входе родителя

Ребёнку после триала — мягкий экран «Попроси родителя продлить» (без вины).

6. Технический стек

  • Next.js 15 (App Router) + TypeScript + Tailwind CSS + shadcn/ui
  • SQLite через Prisma (переезд на Postgres = смена конфига)
  • NextAuth: родитель — email+пароль; ребёнок — профиль + PIN (4 цифры)
  • ЮKassa через API-роуты Next.js; mock-режим для разработки
  • Тесты: Vitest (unit/integration) + Playwright (E2E)

7. Модель данных

Parent        — id, email, passwordHash, createdAt
Child         — id, parentId, name, age, pin, mascotLevel, streak
Subscription  — id, parentId, status, trialEndsAt, paymentMethodId, nextBillingAt
Exercise      — id, type (comic/data/robot), level (1-3), contentJson
Session       — id, childId, startedAt, finishedAt, feedbackEmoji
SessionResult — id, sessionId, exerciseId, accuracy, durationSec, scoresJson
Achievement   — id, childId, badgeKey, earnedAt

8. Экраны (9)

Детские (6)

  • Выбор профиля + PIN
  • Главный: маскот, «Начать сессию», стрик
  • 3 экрана упражнений
  • Итоги сессии: что развивал + эмодзи-фидбек (😍😊😐😕)
  • Карта прогресса / бейджи

Родительские (3)

  • Регистрация / вход
  • Дашборд: 4 шкалы с недельной динамикой, сессии, статус подписки
  • Подписка: оплата ЮKassa, отмена, история

9. Дизайн и качество

  • Светлая тема, акцент #6366F1, секондари #06B6D4, шрифт Inter
  • Детский текст ≥18px, кнопки ≥44px, переходы 200–300ms, WCAG AA
  • Прерванная сессия: сохраняется, продолжение в течение часа
  • Тесты: unit (движок), integration (API, подписка), E2E (happy-path сессии)
